Learn more about how this works:
  • infoWhat is this doing?

    Your text is analyzed using IBM Watson's natural language processing to help you to understand its sentiment, keywords, entities, high-level concepts and more.

  • languageWhat is natural language processing?

    Natural language processing is a field of computer science, artificial intelligence, and computational linguistics concerned with the interactions between computers and human (natural) languages. As such, NLP is related to the area of human–computer interaction.

  • view_moduleUnderstanding the results.

    With Emotion Analysis, you can bring sensitivity to analytics. Emotion Analysis can infer emotions from text samples by leveraging sophisticated machine learning to return confidence scores for anger, disgust, fear, joy, and sadness. Each score indicates the probability that the corresponding emotion is implied by the sample text. Emotion Analysis uses a stacked generalization based ensemble framework to derive the final emotion scores from initial results of several machine learning models. Each lower level model uses a unique combination of machine learning algorithms and language features such as words, phrases, punctuation, and overall sentiments.
